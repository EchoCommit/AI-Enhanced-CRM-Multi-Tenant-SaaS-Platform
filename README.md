# Nexus CRM

Nexus CRM is an AI-first, multi-tenant B2B SaaS platform that unifies customer intelligence, pipeline management, and revenue automation for growing enterprise teams.

## Features
- **Lead Scoring Engine:** Predicts the likelihood of a lead converting to closed-won.
- **Churn Propensity Model:** Flags at-risk deals and accounts using behavioral signals.
- **Deal Close Probability:** Real-time percentage scoring for active deals.
- **Multi-Tenant Architecture:** Secure schema-isolated data environments.
- **Generative AI Workflows:** Automated email sequencing and next-best-action recommendations.

## Getting Started

### Prerequisites
- Python 3.10+
- PostgreSQL
- Node.js & npm (for frontend)

### Setup
1. Clone the repository
2. Install Python dependencies: `pip install -r requirements.txt` (or equivalent)
3. Set up environment variables in `.env`
4. Run migrations and start the server
