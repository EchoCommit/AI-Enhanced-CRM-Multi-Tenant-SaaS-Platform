# Security Policy

## Supported Versions

Please see the table below for supported versions that currently receive security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our platform very seriously. If you discover a security vulnerability within Nexus CRM, please send an e-mail to our security team at `security@nexus-crm.com`. All security vulnerabilities will be promptly addressed.

Please do not report security vulnerabilities through public GitHub issues.

### What to include in your report:
* A detailed description of the vulnerability
* Steps to reproduce the issue
* Potential impact of the vulnerability

We will attempt to acknowledge receipt of your vulnerability report within 48 hours and provide updates as we investigate the issue.
